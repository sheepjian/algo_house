###Chapter 1 Sorting Algorithm

####Definition
A [**sorting algorithm**](http://en.wikipedia.org/wiki/Sorting_algorithm) is an algorithm that puts
elements of a list in a certain order. Popular sorting algorithm can be divided into three categories, 
*simple sort* (insertion sort, selection sort), *efficient sort* (merge sort, heapsort, quicksort), 
*bubble sort* (bubble sort, shell sort, comb sort) and *distribution sort* (counting sort, bucket sort, 
radix sort).
